Shakyamuni a quatre paroles hautement spirituelles :

1. ダレニ デアッテモ、ソノヒトハ アナタノ ジンセイニ アラワレルベキ ヒトデアリ、ケッシテ グウゼンデハ ナイ；
1. Qui que vous rencontriez, c'est la personne qui devait apparaître dans votre vie, ce n'est pas par hasard ;
2、无论发生什么事，那都是唯一会发生的事；
2. Whatever happens, that is the only thing that could have happened;
2. 何が起ころうとも、それは起こるべくして起こった唯一のことである；
2. ナニガ オコロウトモ、ソレハ オコルベクシテ オコッタ ユイイツノ コトデアル；
2. Quoi qu'il arrive, c'est la seule chose qui pouvait arriver ;
3、不管事情开始于哪个时刻，都是对的时刻；
3. No matter when something starts, it is the right moment;
3. 物事がいつ始まろうとも、それは正しい瞬間である；
3. モノゴトガ イツ ハジマロウトモ、ソレハ タダシイ シュンカンデアル；
3. Peu importe quand quelque chose commence, c'est le bon moment ;
4、已经结束的，就已经结束了。
4. What has ended, has already ended.
4. 終わったことは、もう終わっている。
4. オワッタ コトハ、モウ オワッテイル。
4. Ce qui est terminé, est déjà terminé.
